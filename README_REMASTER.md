# AICoverGen Remaster (All-in-One)

Esta versión ha sido simplificada para funcionar en **Google Colab Free** con una sola celda.

## Instrucciones rápidas:
1. Abre el archivo `.ipynb` incluido en Google Colab.
2. Ejecuta la única celda de código.
3. Sube este archivo ZIP (`aicovergen_remaster.zip`) cuando se te pida.
4. Sube tu canción y tu modelo RVC.
5. El cover se generará y descargará automáticamente.

## Mejoras aplicadas:
- **URLs Corregidas:** Se arreglaron los errores 404 de los modelos MDX.
- **Motor Applio:** Inferencia moderna, más estable y rápida.
- **Gestión de VRAM:** Inferencia aislada que libera memoria tras cada uso.
- **Fallback CPU:** Si la GPU se queda sin memoria, el proceso reintenta automáticamente en CPU.
- **Todo en uno:** Instalación, descarga de modelos y ejecución en una sola celda.
